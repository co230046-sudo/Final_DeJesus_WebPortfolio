Wordify C++ Game

Description

Working with classmates, I helped develop Wordify, a text-based Crossword Puzzle game in C++. This project was a hands-on way to apply and demonstrate foundational contept of data structure. The game's structure relies heavily on custom data structures: we used a Linked List to manage and traverse the game levels, a Stack to track the player's lives (hearts), and a Queue to initialize the sequence of rounds within each level. This collaboration is a strong example of my ability to apply core data structures to build a complex application.
